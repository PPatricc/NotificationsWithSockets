#### Firstly run Server.java in your terminal, then launch Client.java, you will be asked to enter your username
![image](https://user-images.githubusercontent.com/86026100/168163255-4c4e8f8c-ac71-44aa-8814-fa9b116bb3cf.png)

![image](https://user-images.githubusercontent.com/86026100/168163196-464e48d1-e879-4f93-a035-31deb825d1c3.png)
